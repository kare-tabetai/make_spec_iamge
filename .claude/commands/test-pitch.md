企画書のテスト生成を実行してください。

## 実行手順

1. 以下のコマンドをBashツールで実行してください：

- 引数が指定されている場合:
```bash
cd /workspace && uv run game-pitch --topic "$ARGUMENTS" --mode test --num-pitches 2
```

- 引数が空の場合（デフォルト）:
```bash
cd /workspace && uv run game-pitch --topic "お題:「不自由」" --mode test --num-pitches 2
```

2. 実行が完了したら、出力ディレクトリ（`output/` 配下の最新フォルダ）を確認してください。

3. 生成された企画書（`pitch_*/pitch.md`）の内容を読み取り、以下の観点でサマリーを報告してください：
   - 生成された企画書の数
   - 各企画書のタイトルとコンセプト概要
   - 画像生成の成否
   - エラーがあった場合はその内容

## 引数について

- 引数にはトピック（お題）を指定してください
- 例: `/test-pitch お題:「不自由」`
- 例: `/test-pitch テーマ:「時間」を使ったパズルゲーム`
- 引数が空の場合は、デフォルトのトピック `お題:「不自由」` を使用してください
