
- GoogleResearchAgent、DuckDuckGoResearchAgentでは与えられたtopicについて調査する。市場調査等は不要
- BrainstormAgentはアイデア創出法ごとに別のサブエージェントに分ける
- アイデア創出法にマンダラートとアイデアしりとりを追加する
- ExpansionAgentでターゲットは書かない
- 生成リクエストで指定した引数の内容がわかるようにログファイルを別途出力する
- アイデアを出すエージェントの温度の設定を高めにする
- 画像ペラ１企画書の説明が薄過ぎてよくわからないため、プロンプトを調整してコンセプトとコアアイデア、コアメカニクスを表示するようにする
- テストで使われているtopicの「宇宙を舞台にした孤独なロボットの旅」が曖昧過ぎて評価しづらいためテスト実行の際は「不自由」を用いるようにする
- テスト実行の際に３枚も画像企画書が生成されるのはトークン使用料の無駄になるため２枚の生成にする
